Latest Project

In my matric year I created a maze solving computer game but unlike most maze games this game required the player to collect keys in the correct order to continue onto the next level. The keys were the characters "K","E","Y" and were located on different levels, yes each maze was multistoried. This game allowed movement through the w,a,s,d keys as well as choosing the difficuly of game play ranging from easy to a mode where only the immediate spaces surrounding the player were visible while the rest were "Blacked out".The point of the game was to get the fastest solve time which was then stored in a database and displayed on a leaderboard.
